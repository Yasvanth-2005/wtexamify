# 🎯 Features Overview
### 👩‍🎓 Student Panel

- ✅ **RGUKT Email Login**: Students login using their RGUKT email ID.
- 🧾 **View Started Exams Only**: Students can only see exams that have been started by the teacher.
- 🚫 **Anti-Cheating Protection**: Reload, back navigation, new tabs, and copy/paste are blocked.
- ⚠️ **Violation Handling**: Triggers a "Copied" box that requires a secret teacher code to continue.
- ⏳ **Auto Submission**: If the code isn't entered, the exam is auto-submitted.
- 🔄 **Manual Question Refresh**: Teachers can refresh questions live if suspicious behavior is detected.
- 🧠 **AI Evaluation**: Submitted exams are scored by AI with marks and remarks shown instantly.

### 👨‍🏫 Teacher Panel

- 🔐 **Normal Email Login**: Teachers can log in using any valid email address.
- 📝 **Create Exams**: Supports creation of Internal, External, and 10/15-bit Viva exams.
- ▶️ **Start/Stop Exams**: Full control over when to begin or end an exam session.
- 📡 **Live Monitoring**: Teachers can manually refresh student questions in real time if needed.
- 🧾 **Answer Script Access**: View student submissions as PDFs along with AI-evaluated scores.
- ⬇️ **Bulk Download**: Download all student answer scripts with a single click.
- 🔒 **Secret Code Control**: Provides teachers with a secret code to unlock “Copied” warnings during exams.

### 🔐 Exam Integrity

- 🚫 **No Reload / Back / Tab Switch**: Students are blocked from reloading the page, going back, or opening new tabs.
- ❌ **Copy/Paste Disabled**: Common cheat actions like Ctrl+C / Ctrl+V are restricted.
- ⚠️ **Copied Warning Box**: Violations show a warning requiring a secret teacher code to continue.
- ⏳ **Auto Submission**: If the code is not entered in time, the exam is automatically submitted.
- 🔄 **Live Control by Teacher**: Teachers can refresh a student's questions on suspicion of misconduct.

### 🧠 AI-Powered Evaluation (Special feature)

- 🤖 **Instant Scoring**: Exams are automatically scored by the AI after submission.
- 🗂️ **Subjective & Objective Support**: Handles various question types effectively.
- 💬 **Smart Remarks**: AI provides feedback/remarks alongside scores for better understanding.
- 📄 **PDF Answer Sheets**: Evaluated answer scripts are available as downloadable PDFs.
- 📥 **One-Click Download**: Teachers can download all answer sheets together for review or recordkeeping.
